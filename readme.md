# 飛行日誌分析儀 (Flight Log Analyzer)

## 專案描述

一個用於分析飛行日誌 CSV 檔案的整合平台。它能夠將數據透過互動式圖表進行視覺化，並提供 RMA (Return Merchandise Authorization) 檢測記錄的工具。此工具旨在幫助技術人員快速診斷飛行問題、記錄發現，並生成可保存的分析報告。

## 主要功能

*   **日誌檔案解析**: 支援上傳並解析飛控產生的 `.csv` 格式日誌檔案。
*   **互動式數據圖表**:
    *   動態顯示多個飛行參數群組，如電源系統、飛行控制、撞擊分析等。
    *   所有圖表的時間軸同步，可透過主時間軸進行縮放與平移。
    *   支援平滑曲線顯示，提升圖表可讀性。
*   **飛行軌跡視覺化**:
    *   在 2D 地圖 (Leaflet) 上顯示飛行軌跡，並標示起點與終點。
    *   在 3D 地球 (Cesium) 上呈現立體的飛行路徑，提供更直觀的高度與地理位置資訊。
    *   圖表與地圖時間同步，滑鼠懸停在圖表上時，地圖會顯示對應時間點的飛行器位置。
*   **RMA 檢測工具**:
    *   提供欄位以記錄 RMA 單號、產品型號等資訊。
    *   內建電子簽名功能，方便檢測人員簽署。
    *   提供筆記區塊，供使用者記錄分析過程中的觀察與結論。
*   **報告匯出**: 可將包含 RMA 資訊、圖表、軌跡地圖與筆記的完整分析頁面儲存為單一的 HTML 檔案，便於歸檔與分享。
*   **客製化介面**: 支援深色與淺色模式切換，以適應不同的工作環境。

## 使用流程

1.  **上傳日誌**: 點擊「上傳並分析日誌」按鈕，選擇您的 `.csv` 檔案。
2.  **等待分析**: 系統會自動解析檔案。完成後，您將在左側看到「日誌資訊」，右側看到「時間軸調整」與預設的圖表。
3.  **檢視圖表**: 在左側「日誌檢查功能」區塊，點擊按鈕（如「電源與電力系統」、「飛行控制系統」）來顯示或隱藏對應的圖表群組。
4.  **互動操作**:
    *   在「時間軸調整」區塊，拖曳選取框以縮放特定時間範圍。
    *   將滑鼠游標懸停在任何圖表上，所有圖表和 2D 地圖上的飛行器位置將會同步顯示該時間點的數據。
5.  **記錄資訊**:
    *   在左上方的「RMA 資訊」區塊填寫相關單號與型號。
    *   點擊「點此簽名」完成簽署。
    *   在「筆記」區塊記錄您的分析發現。
6.  **儲存報告**: 填寫完所有資訊後，點擊「另存為 HTML 檔案」，即可將當前頁面完整保存下來。

## 本地部屬流程

此應用程式是使用 React 和 TypeScript 開發的前端專案，需要 Node.js 環境來進行本地開發與運行。

### 前置需求

*   [Node.js](https://nodejs.org/) (建議使用v20.19.5 LTS 版本)
*   npm (通常會隨 Node.js 一併安裝)

### 步驟

1.  **安裝依賴套件**:
    打開您的終端機 (Terminal) 或命令提示字元 (Command Prompt)，並切換到專案的根目錄，然後執行以下指令來安裝所有必要的套件：
    ```bash
    npm install
    ```

2.  **啟動開發伺服器**:
    安裝完成後，執行以下指令來啟動本地開發伺服器：
    ```bash
    npm run dev
    ```
